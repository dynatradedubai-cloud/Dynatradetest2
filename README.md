# Dynatrade Parts Portal v5

## Features
- Real IP detection using https://api.ipify.org for IP-based login validation
- Excel-based Username/Password/IP authentication
- Clear Cart works on first click
- 'Add to Cart' button in last column with header
- Separate Customer and Admin pages
- Campaign file download link for customers
- Multi-format campaign upload

## Setup Instructions
1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   streamlit run dynatrade_parts_portal_v5.py
   ```
